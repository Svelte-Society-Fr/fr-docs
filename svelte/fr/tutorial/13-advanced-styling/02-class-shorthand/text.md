---
title: Directive de classe raccourcie
---

Souvent, le nom de la classe sera le même que celui de la variable dont elle dépend :

```html
<div class:big={big}>
	<!-- ... -->
</div>
```

Dans ce cas-là, nous pouvons utiliser la forme raccourcie :

```html
<div class:big>
	<!-- ... -->
</div>
```
